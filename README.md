# 📊 Superstore Sales Dashboard (Microsoft Excel)

## Project Overview
This project presents an interactive Sales Dashboard built in Microsoft Excel using the Superstore dataset. The dashboard enables users to analyze sales performance through dynamic filtering, KPI cards, Pivot Tables, Pivot Charts, and Slicers.

## Key Highlights
- Analyzed **9,994 sales records** from the Superstore dataset.
- Created interactive KPI Cards for:
  - Total Sales
  - Total Profit
  - Total Orders
- Built Pivot Tables to summarize business data.
- Designed Pivot Charts for visual analysis.
- Added Slicers for Category, Region, and Segment to enable dynamic filtering.
- Developed a clean and user-friendly dashboard layout.
